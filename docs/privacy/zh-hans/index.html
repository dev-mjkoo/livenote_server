隐私政策

本政策说明 LiveNote 如何保护用户隐私并处理相关数据。

最后更新: 2026年2月24日

1. 应用信息

应用名称：LiveNote

联系方式：help@lvnt.my

2. 我们收集的数据

应用可能通过 Firebase Analytics 收集匿名统计信息。

  • 收集内容：应用使用模式（功能使用情况、使用频率等）
  • 收集目的：改进服务并提升用户体验
  • 说明：不会收集可识别个人身份的信息
  • 存储位置：Google Firebase 服务器

3. 用户选择权

您可以随时在应用设置中关闭分析数据收集。

4. iCloud 同步

您的笔记、链接和分类通过 Apple iCloud 进行同步。

  • 开发者无法访问存储在 iCloud 中的个人数据。
  • 该数据受 Apple 隐私政策约束。

5. 法律合规（GDPR 与韩国个人信息保护法）

LiveNote 遵守欧盟《通用数据保护条例》（GDPR）及韩国《个人信息保护法》。

您享有以下权利：

  • 请求访问和更正个人信息
  • 请求删除数据
  • 请求限制处理或提出反对
  • 撤回数据收集同意

您可通过开发者联系方式提交请求。

6. 联系方式

隐私相关咨询：dev.livenote@gmail.com
